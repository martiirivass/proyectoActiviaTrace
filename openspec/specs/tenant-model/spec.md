## Requirements
### Requirement: Tenant entity exists with required attributes
The system SHALL provide a `Tenant` model representing an institution with the following attributes:
- `id`: UUID, primary key, generated by database
- `nombre`: text, NOT NULL, human-readable institution name
- `slug`: text, NOT NULL, unique, URL-safe identifier for subdomain routing
- `estado`: enum (Activa, Inactiva, Suspendida), NOT NULL, default 'Activa'
- `configuracion`: JSONB, NOT NULL, default '{}', tenant-specific settings
- `created_at`: timestamp with timezone, NOT NULL, auto-set on insert
- `updated_at`: timestamp with timezone, NOT NULL, auto-set on insert and update
- `deleted_at`: timestamp with timezone, NULLABLE, set on soft delete

#### Scenario: Tenant creation with all required fields
- **WHEN** a new Tenant is inserted with nombre, slug, and estado
- **THEN** the row is created with generated UUID id, auto-populated timestamps, and default configuracion

#### Scenario: Tenant slug uniqueness enforced
- **WHEN** attempting to insert a second Tenant with the same slug
- **THEN** database raises unique constraint violation

#### Scenario: Tenant estado defaults to Activa
- **WHEN** a Tenant is created without explicit estado
- **THEN** the estado column value is 'Activa'

#### Scenario: Tenant configuracion defaults to empty JSON object
- **WHEN** a Tenant is created without explicit configuracion
- **THEN** the configuracion column value is '{}'

### Requirement: Tenant lifecycle operations
The system SHALL support tenant lifecycle operations:
- Create tenant (Activa by default)
- Update tenant nombre, estado, configuracion
- Soft delete tenant (sets deleted_at, changes estado to Suspendida)
- List active tenants (excludes deleted)

#### Scenario: Soft delete sets deleted_at and changes estado
- **WHEN** tenant is soft-deleted
- **THEN** deleted_at is set to current timestamp and estado becomes 'Suspendida'

#### Scenario: List active tenants excludes soft-deleted
- **WHEN** querying active tenants
- **THEN** only tenants with deleted_at IS NULL are returned
